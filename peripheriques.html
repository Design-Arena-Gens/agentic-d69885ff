Les périphériques

Les périphériques permettent à l’ordinateur d’interagir avec le monde extérieur. Ils sont reliés via des bus d’E/S et pilotés par des contrôleurs dédiés.

On distingue les périphériques d’entrée (clavier, capteurs), de sortie (écran, imprimante) et mixtes (disques, interfaces réseau).

Classification

Type Exemples Caractéristiques
Entrée Clavier, souris, scanner, micro Convertissent les signaux physiques en données numériques.
Sortie Moniteur, haut-parleurs, projecteur Transforment les données numériques en signaux perceptibles.
Stockage SSD, HDD, clés USB Assurent la persistance des données avec des vitesses variées.
Communication Carte réseau, Bluetooth Permettent l’échange de données avec d’autres systèmes.

Bus et contrôleurs

  • USB : interface universelle à chaud avec alimentation intégrée.
  • PCI Express : bus interne haute performance pour cartes graphiques ou SSD NVMe.
  • I²C / SPI : bus série pour microcontrôleurs et capteurs.

Chaque périphérique est associé à un contrôleur qui gère les transferts de données. Les interruptions matérielles (IRQ) notifient le CPU lorsqu’une opération est terminée.

Gestion par le système d’exploitation

Pilotes (drivers) Composants logiciels qui traduisent les commandes système en instructions spécifiques au périphérique. Ils offrent une interface standardisée aux applications tout en assurant la sécurité et l’isolation.

Le système d’exploitation utilise des files d’attente d’E/S, des tampons (buffering) et des transferts DMA (Direct Memory Access) pour optimiser le débit et la latence.
