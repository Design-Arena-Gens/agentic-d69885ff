Le processeur (CPU)

Le CPU est l’unité de traitement générale de l’ordinateur. Il exécute les instructions machine, coordonne les opérations et gère les échanges avec les autres composants.

La performance d’un processeur dépend de son architecture interne (pipeline, cœurs, cache), de sa fréquence et de l’efficacité des instructions.

Les registres contiennent les données manipulées instantanément. Le cache hiérarchisé minimise les temps d’accès à la mémoire principale.

Pipeline d’instructions

Le pipeline permet de découper l’exécution d’une instruction en étapes successives (fetch, decode, execute, memory, writeback) exécutées en parallèle sur plusieurs instructions distinctes.

  • Superscalaire : plusieurs instructions peuvent entrer dans le pipeline dans le même cycle.
  • Out-of-order : réordonnancement dynamique pour exploiter le parallélisme.
  • Prédiction de branche : spéculation sur l’issue des tests pour éviter les bulles.
