Glossaire

Retrouvez ici les principaux termes utilisés dans ce cours. Chaque définition vous aidera à consolider votre compréhension de l’architecture des ordinateurs.

Termes clés

Bit

Plus petite unité d’information binaire, pouvant prendre la valeur 0 ou 1.

Bus

Canal de communication permettant de transférer des données entre composants matériels.

Cache

Mémoire rapide destinée à stocker temporairement des données pour réduire les temps d’accès.

Kernel

Cœur du système d’exploitation qui gère les ressources et assure l’interface avec le matériel.

Pipeline

Mécanisme permettant de traiter plusieurs instructions simultanément en les découpant en étapes.

RAM

Mémoire vive volatile utilisée pour stocker les programmes et données en cours d’exécution.

ROM

Mémoire en lecture seule contenant généralement le microprogramme ou le firmware initial.

ISA

Instruction Set Architecture : ensemble des instructions que le processeur peut exécuter.

MMU

Memory Management Unit : composant chargé de la traduction des adresses virtuelles en adresses physiques.

Thread

Flux d’exécution léger partageant le même espace mémoire qu’un processus.

Virtualisation

Technique permettant d’exécuter plusieurs systèmes isolés sur un même matériel physique.

DMA

Direct Memory Access : transfert direct de données entre périphériques et mémoire sans passer par le CPU.
